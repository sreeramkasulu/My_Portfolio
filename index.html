Recent Projects

Project 1

OTP Generator using Gmail SMTP Server

I developed an OTP (One-Time Password) generator using Python and Gmail’s SMTP server to facilitate secure email-based OTP verification. The project includes:

Email Validation: Ensures the entered email belongs to common domains like Gmail, Yahoo, Outlook, etc.

OTP Generation: A 6-digit OTP is randomly generated.

OTP Delivery: The OTP is sent via email using the SMTP server.

Resend Option: Users can opt to resend the OTP either to the same or a new email address if verification fails.

This project demonstrates secure email handling and Python integration with SMTP for real-world applications.

Project 2

Constitution of India Chatbot

I developed a conversational AI chatbot that answers questions related to the Constitution of India. The chatbot utilizes natural language processing techniques, combining LangChain for document loading and splitting, SentenceTransformers for semantic search, and GPT-Neo for generating human-like responses. The chatbot allows users to interactively query content from the Constitution, providing accurate answers based on relevant sections of the document.

Technologies Used: LangChain: Document loading and text processing. SentenceTransformers: Embedding generation for document similarity. GPT-Neo: Language model for text generation. Transformers & PyPDFLoader: For document parsing and question answering.

Project 3

Telugu Character Recognition

I Developed a machine learning-based system for recognizing Telugu characters using deep learning techniques. The project involved training a convolutional neural network (CNN) model on a labeled dataset of Telugu characters to accurately identify and classify them. Key Features: Built a web application interface using Flask to upload and recognize Telugu characters in real-time. Used TensorFlow to create and train a CNN model, achieving high accuracy in character recognition. Integrated a labeled dataset of Telugu characters for model training and evaluation. The final model is deployed for practical use, allowing users to test Telugu character inputs and receive classification results. Technologies Used: Python, Flask, TensorFlow, Keras HTML, CSS, JavaScript for web interface Model deployment via Flask with interactive UI
